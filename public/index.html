WriteLens

A fast second reader for drafts that sound finished before they are.

What it catches

WriteLens reads a draft for the things a good editor marks in the margin: foggy claims, weak structure, generic persuasion, and the first sentence that should become specific. It is free to try in the browser, with no account and no upload beyond the scoring request.

For tools and teams

The same reader is available as a small API when you need a writing-quality signal inside an editor, classroom workflow, review queue, or publishing pipeline.

POST /v1/score

curl -X POST https://writelens.ivixivi.workers.dev/v1/score \ -H "Content-Type: application/json" \ -H "Authorization: Bearer wl_yourkey" \ -d '{"text":"your text here"}'

Returns:

{ "clarity": 4, "persuasiveness": 3, "structure": 5, "factuality_concerns": ["unsubstantiated claims about 'experts'"], "overall": 4, "suggestions": [ "Replace 'various factors' with the actual factors", "Cite specific experts or remove the appeal to authority" ], "char_count": 312 }

Why
